# 😊 Emotion Detection using NLP

## 📌 Project Overview
This project is a Natural Language Processing (NLP) based Emotion Detection system that classifies text into different emotions such as joy, sadness, anger, fear, love, etc.

The model uses Machine Learning techniques with TF-IDF vectorization to understand and predict the emotional tone of input text.

---

## 🎯 Objective
To build a model that can detect emotions from text and can be used in applications like:
- Chatbots with emotional understanding
- Sentiment analysis systems
- Social media text analysis
- Customer feedback analysis

---

## 🧠 Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- NLP (Natural Language Processing)
- TF-IDF Vectorizer
- Pickle (for model saving)
